Privacy Policy

Last updated: March 30, 2025

This browser extension (“we”, “our”, or “the extension”) helps users export and restore selected open tabs using a short 6-character code. We are committed to protecting your privacy and ensuring a safe, transparent experience.

What Data We Collect

We only collect the following information, and only when you choose to export tabs:

  • Tab title
  • Tab URL
  • Tab favicon

We do not collect:

  • Your identity (name, email, IP address)
  • Browsing history not selected by you
  • Cookies or content of web pages

How We Use Your Data

We use the tab information you choose to export solely to provide the core functionality of this extension. Specifically:

  • The selected tab titles, URLs, and favicons are sent securely to our backend via HTTPS.
  • This data is stored in a protected database and linked only to a randomly generated 6-character code.
  • The code is displayed to you, allowing you to reopen or share your tab session later.

You control when to export and share this code. No automatic syncing or background tracking is involved.

Clipboard Access

We request clipboard permissions to improve usability:

  • clipboardWrite lets you copy the generated code with one click.
  • clipboardRead is used when you click “Paste” to restore a session.

Clipboard access is only triggered through visible user actions.

Data Retention

Tab data exported through the extension may be stored on our servers for an indefinite period of time. While we may implement automatic cleanup in the future, currently there is no fixed expiration date for stored data. Each export is stored anonymously and can only be accessed using the unique 6-character code generated at the time of export. We do not associate this data with any personal identifiers. If you wish to have your exported data removed, please contact us with the corresponding code, and we’ll manually delete it.

Data Sharing

We do not share, sell, or disclose any user data to third parties under any circumstances. All exported tab data remains private and is only accessible through the unique 6-character code generated during the export process. We do not use this data for analytics, marketing, or profiling, and we do not integrate with any third-party services that access or process your information.

Security

  • All communication is encrypted (TLS 1.2+).
  • Data is validated, stored securely, and access-controlled.
  • Only tab metadata is handled — no page content, cookies, or identity data.

Your Consent

By using this extension, you agree to:

  • Voluntary data transfer of selected tabs
  • Clipboard usage triggered by your clicks
